Methods 静态类¶

  • 命名空间：Autolabor.PM1

  • 源文件：Methods.cs

  • 库文件：pm1_sdk_native.dll、PM1.SDK.Net.dll

提供一组用于控制机器人的静态方法。

  • 属性

属性¶

名字 描述
Parameters 获取参数索引器。
Odometry 获取里程计读数。
State 获取或设置底盘工作状态。
PhysicalTarget 设置物理模型下的目标运动状态。
WheelsTarget 设置差动模型下的目标运动状态。
VelocityTarget 设置运动模型下的目标运动状态。
Paused 查询、设置或解除暂停。
