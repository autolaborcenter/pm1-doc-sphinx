  • 源文件：Methods.cs

参数签名 描述

Drive(double v, double w, double spatium, out double progress)¶

定义¶

public static void Drive(double v, double w, double spatium, out double progress)

参数¶

  • v - 线速度（米/秒）

  • w - 角速度（弧度/秒）

  • spatium - 空间约束（参见 CalculateSpatium）

  • progress - 进度

返回值¶

（无）
