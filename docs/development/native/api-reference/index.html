PM1 原生 API 参考¶

API 约定¶

  • 坐标系

    • 所有坐标系为右手性的平面直角坐标系；

    • 机器人初始化后或里程计清零后，认为机器人坐标系和里程计坐标系重合；

    • 机器人坐标系 X 轴正方向向机器人前方，Y 轴正方向向机器人左方；

  • 调用顺序

    • 如无特殊说明，函数需要在初始化之后、断开连接之前调用；
