autolabor::pm1::go_arc_wt¶

声明于头文件 pm1_sdk.h

autolabor::pm1::result<void>
autolabor::pm1::go_arc_wt(
    double w,
    double r,
    double t,
    double *progress = nullptr);

按圆弧行驶。

参数¶

  • w - 角速度

  • r - 转弯半径

  • t - 时间约束

  • progress - 进度，若非空，可在阻塞期间获取实时进度，取值范围 [0,1]

返回值¶

若失败，result.error_info 保存错误信息。

注意¶

这是一个动作，将阻塞当前线程直到执行完毕。可以通过调用 cancel_action 取消。
