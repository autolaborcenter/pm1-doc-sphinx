约定¶

  • 参数

参数类型 单位 取值范围 正方向
线速度 m/s 实数 前
角速度 rad/s 实数 逆时针
轨迹弧长 m 非负实数 -
轨迹圆心角 rad 非负实数 -
转弯半径 m 实数 逆时针
时间段 s 非负实数 -

  • 坐标系

    • 所有坐标系为右手性的平面直角坐标系；

    • 机器人初始化后或里程计清零后，认为机器人坐标系和里程计坐标系重合；

    • 机器人坐标系 X 轴正方向向机器人前方，Y 轴正方向向机器人左方；

  • 调用顺序

    • 如无特殊说明，函数需要在初始化之后、断开连接之前调用；
