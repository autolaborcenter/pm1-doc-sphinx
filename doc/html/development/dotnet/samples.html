PM1 .Net API 使用范例¶

在 Visual Studio 环境下使用 C# API 开发¶

  • .Net DLL (PM1.SDK.Net.dll)

  • Native DLL (pm1_sdk_native_debug.dll、pm1_sdk_native.dll)

注意：程序运行时还需要将 native DLL 文件拷贝至运行目录下。
