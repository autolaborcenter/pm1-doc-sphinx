autolabor::pm1::drive_spatial¶

声明于头文件 pm1_sdk.h

autolabor::pm1::result<void>
autolabor::pm1::drive_spatial(
    double v,
    double w,
    double s,
    double *progress = nullptr);

控制机器人按空间约束运行指定动作。

参数¶

  • v - 线速度

  • w - 角速度

  • s - 空间尺度约束，见 calculate_spatium

  • progress - 进度，若非空，可在阻塞期间获取实时进度，取值范围 [0,1]

返回值¶

若失败，result.error_info 保存错误信息。

参考¶

阅读控制机器人行驶了解控制方式。
